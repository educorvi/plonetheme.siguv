================
plonetheme.siguv
================

A Plone 5.2 Theme for the siguv Berufsgenossenschaften


Installation
------------

Install plonetheme.siguv by adding it to your buildout::

    [buildout]

    ...

    eggs =
        plonetheme.siguv


and then running ``bin/buildout``
